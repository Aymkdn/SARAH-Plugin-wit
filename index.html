Auteur: Aymeric (Twitter GitHub Blog)

Ce plugin permet d'interagir avec https://wit.ai/ pour effectuer des commandes naturelles.

Vous devez configurer ce plugin en entrant votre Server Access Token que vous trouverez après avoir créé une instance sur le site de Wit.ai. Après ça il faut enseigner des phrases au site web. Par exemple "pourrais-tu allumer la télé" ou "je materais bien la télé" comme phrase pour un Intents qu'on doit appeler par {plugin}_{action} donc ici on l'appelle freebox_tv_on.

Enfin il faut aller bidouiller dans le fichier wit.js pour associer les mots-clés retournés par Wit.ai (par exemple freebox_tv_on) à des actions (par exemple allumer la Freebox).
